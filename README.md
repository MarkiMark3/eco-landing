# The Eco-cosmetics page

The Eco-cosmetics page is a project that focuses on simple yet captivating front-end development and responsive design.

## Technologies Used

- HTML
- SCSS
- Stylelint

## How to Run This Project Locally

1. Clone this project from GitHub repository.
2. Ensure **Node version** should be 14.
3. Run the following command:

- npm install
- npm start
